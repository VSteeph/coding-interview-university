
# Syntaxe

## Class Creation
On peut créer une classe ou element puis lui donner toutes les données voulues avec le format suivant

```
SomeClass a = new SomeClass()
{
var1 = value,
var2 = value,
var3 = value,
};
```

au lieu de faire
```
SomeClass a = new SomeClass();
a.var1 = value;
a.var2 = value;
a.var3 = value;
```
